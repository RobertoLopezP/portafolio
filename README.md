## 💼 Proyectos Destacados

### 🌆 Análisis de Movilidad Urbana y Productividad Económica en LATAM
**Bootcamp Data Analyst - TripleTen (2026)**

Proyecto integral de análisis que explora la relación entre congestión vehicular y productividad económica en ciudades latinoamericanas.

**🎯 Objetivo:** Identificar patrones entre movilidad urbana y PIB per cápita para apoyar decisiones estratégicas en infraestructura de transporte.

**🔧 Proceso:**
- 🧹 Limpieza y estandarización de múltiples datasets de tráfico y economía
- 🔄 Transformación y unificación de datos usando Python (Pandas, NumPy)
- 📊 Análisis exploratorio y correlación de variables clave
- 📈 Visualización de insights con Matplotlib y Seaborn

**✨ Resultados:**
- Identificación de ciudades prioritarias para inversión en infraestructura
- Generación de insights accionables sobre la relación movilidad-productividad
- Dataset unificado y documentado listo para análisis adicionales

**🛠️ Stack Tecnológico:**
`Python` `Pandas` `NumPy` `Matplotlib` `Seaborn` `Data Cleaning` `Data Visualization`
